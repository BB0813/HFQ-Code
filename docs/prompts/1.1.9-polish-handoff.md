## 2. 实现指引（按模块）

### 2.1 Terminal（P1）

路径：`apps/desktop/renderer/src/features/terminal/*` · `store/terminal-store.ts` · 现有 `pty:*` IPC。

建议：

1. `bootstrap`/`refresh` 后：若 `activeId` 失效 → 选第一个存活或 `null` + 空态「无终端会话」。  
2. 多 tab：标签显示 shell/label；关闭 = kill；切换 = setActive + 复用 xterm 实例策略（保持现有「切页不清屏」）。  
3. workspace 切换后：与 main 的 killAll 行为对齐 — UI 应 `refresh` 清空陈旧列表（订阅 workspace 变更若尚未接）。  
4. 文案中文；错误 toast 短句。

### 2.2 Changes（P2–P3）

路径：`features/changes/ChangesPanel.tsx`。

建议：

1. Commit 区：staged 文件计数 + 列表摘要（可截断）；`message.trim()` 空 → 禁用或 toast。  
2. 键盘：保持 j/k、↑↓；**contenteditable/input/textarea 内 return**（已有逻辑则只补洞）。  
3. 「请 Agent 修改」：确认 `setComposerDraft` + 路由到 `/chat`（若尚未 navigate，补上）；**禁止** auto-send。  
4. 无 git 仓库 / 无 workspace：友好空态，不抛生硬英文。

### 2.3 Tasks（P4）

路径：`features/tasks/TasksPanel.tsx`。

建议：

1. Goal：按 `parentTaskId` 缩进或嵌套列表（**不要**新 goals.db）。  
2. 子会话：打开时 push parentStack；返回 pop + `selectSession`。  
3. `listSpawnAttempts` / `errorCode` 展示保持中文映射；失败可复制。  
4. 不把完整 Goal OS（验收/阻塞工作流产品）塞进本版。
