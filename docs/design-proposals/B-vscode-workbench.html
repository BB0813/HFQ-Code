方案 B 要点

  • 借鉴：VS Code 活动栏 + 资源管理器 + 编辑区标签 + 底栏面板 + 右侧 SCM/改动。
  • 主路径：文件与终端权重更高；Agent 是中心标签之一，而非唯一中心。
  • 适合：习惯 IDE 的用户、需要长期看文件树 / 跑测试。
  • 成本：面板状态最多，最吃前端框架与布局引擎；PTY 底栏是自然落点。
